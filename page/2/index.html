新手投资体系构建：如何构建完整的投资逻辑闭环

发表于 2026-01-29 更新于 2026-01-30 分类于 投资学习笔记

很多投资者在学习投资时，往往觉得知识点杂乱，难以形成一个完整的闭环。

一个成熟的投资体系不仅仅是“看好买入，坐等上涨”，它需要修补逻辑漏洞，建立从宏观到交易执行的完整系统。为了将零散的知识修补成一个可执行的系统，我们需要按顺序补充以下四个维度的知识地图。

第一部分：知识地图构建

1. 宏观层：从“看数据”进阶到“看周期” 🌏

1.1 先行指标（预判未来）

这是最早发出的信号，通常领先股市 3-6个月。

  • 资金面：重点关注 社融增量（尤其是企业中长期贷款）。资金是经济的血液，只要企业开始借钱扩产，经济迟早会热。

  • 订单面：关注 PMI新订单 - PMI产成品库存 的差值。差值扩大，说明供不应求，是制造业复苏的最强信号。

  • 市场面：国债收益率。作为最聪明的钱，债市利率的反弹往往意味着经济见底。

1.2 高频指标（验证现在）

填补月度数据的真空期，按周甚至按天把脉。

  • 生产端：高炉开工率、全钢胎开工率。

  • 流通端：整车货运流量指数（卡车跑得快不快）。

1.3 滞后指标（确认过去）

  • GDP、CPI、企业财报。它们的作用不是预测，而是确认之前的预判是否正确，往往也是利好兑现、考虑止盈的时刻。

2. 中观层：从“原料价格”进阶到“行业景气度” 🏭

  • 误区：只盯着原料成本（如大宗商品价格）。
  • 进阶：看行业的供需格局和商业模式。

核心知识点：行业生命周期与供需分析

必学内容：

  • 价格传导机制：上游涨价，中游能不能把价格传给下游？（例如：面粉涨价，面包能不能涨价？如果不能，面包厂利润就会被挤压）。

  • 波特五力模型：判断一个行业是不是好生意（比如白酒不需要大宗商品，且拥有极强定价权，这比强周期股更值得长期持有）。

关于债券久期：

  • 衰退期：利率下行，长债（10年以上）涨幅最大，因此是优势资产。

  • 过热期/滞胀期：利率上行或高位震荡，长债风险大，短债（1-3年）抗跌性更好。

关于商品分化：

  • 复苏期：工业需求回升，有色金属（铜、铝）表现最好。

  • 衰退期：需求疲软，谷物等必需消费品属性的商品相对抗跌。

3. 微观层：从“挑选标的”进阶到“精准估值” 🔍

  • 误区：看到股价低就觉得是低点，容易掉入“价值陷阱”。

  • 进阶：通过财务分析确认好公司，通过估值建模确认好价格。

在选定好赛道后，我们必须拿起显微镜观察具体的公司。微观分析的核心只有两个任务：一是通过财报分析排除烂公司（排雷），二是通过估值分析算出好价格（定价）。

3.1 财报排雷——三张表的实战看法

财报是公司的体检报告。我们不需要像会计师那样算平每一分钱，作为投资者，我们要看的是“这生意赚不赚钱”以及“这钱是不是真的”。

A. 资产负债表 (看家底)

  • 应收账款：如果 应收账款增速 > 营收增速，说明公司为了冲业绩在“赊销”，或者产品卖不动。这是利润造假的高发区。

  • 商誉：商誉占总资产比例过高（如>20%）非常危险。一旦业绩不达标，商誉减值会吞噬巨额利润。

  • 资产负债率：一般不应超过 60%。如果账上没钱，还欠了一屁股债（短期借款高），随时会破产。

B. 利润表 (看面子)

  • 毛利率：这是公司的护城河。长期毛利率下跌，说明行业竞争加剧，公司不行了。

  • 扣非净利润：一定要看“扣除非经常性损益后的净利润”，排除卖楼或补贴带来的一次性收益。

C. 现金流量表 (看日子) 🌟最重要的表

  • 净现比：经营现金流净额 / 净利润 应该 > 1。

  • 如果你赚了100块利润，但只收回10块钱现金（净现比0.1），说明利润全是“白条”。连续多年净现比小于1，大概率造假。

3.2 估值建模——四大“定价尺子”

好公司也怕买贵了。不同的行业，必须用不同的尺子（估值方法）去量。

1. PE (市盈率) —— 盈利稳定的“回本计算器”

  • 公式：$$PE = \frac{股价 (P)}{每股收益 (EPS)}$$

  • 适用：消费、医药、食品（弱周期、盈利稳定）。

  • 避坑：不适用于周期股（券商、有色），因为周期股往往“高PE是底，低PE是顶”。

2. PB (市净率) —— 破产清算的“家底折扣”

  • 公式：$$PB = \frac{股价 (P)}{每股净资产 (B)}$$

  • 适用：银行、券商、保险、钢铁、煤炭（资产实、重资产）。

3. PEG (市盈率相对盈利增长比率) —— 成长股的“加速度计”

  • 公式：$$PEG = \frac{PE}{G (未来3年预估增长率)}$$

  • 适用：科技股、新能源。

  • 标准：PEG < 1 低估；PEG > 1.5 高估。

4. DCF (自由现金流折现) —— 投资的“终极奥义”

  • 公式：$$Value = \sum \frac{CF_t}{(1+r)^t}$$

  • 适用：水电、高速公路（现金流极其稳定）。

  • 核心逻辑：今天的钱比明天的钱值钱；真金白银比账面利润值钱。

💡 实战操作流程

  1. 先排雷：净现比 < 1？应收账款暴雷？-> 直接 Pass。

  2. 后定价：

    • 银行/周期股 -> 看 PB 分位点。

    • 消费/白马股 -> 看 PE 分位点。

    • 芯片/AI -> 看 PEG。

4. 策略层：从“交给时间”进阶到“仓位管理” 🛡️

  • 误区：买了就不管，盲目相信“时间是玫瑰”。

  • 进阶：建立仓位管理和风控体系，决定能不能睡好觉。

4.1 仓位管理的数学原理——凯利公式

核心作用是算出：每次下注多少比例的本金，能让长期收益最大化，且永远不会破产。

$$f^* = \frac{bp - q}{b}$$

  • f*：下注比例（仓位）

  • b：赔率（盈亏比，如赚30%亏10%，则b=3）

  • p：胜率（成功概率）

  • q：败率（1 − p）

⚠️ 实战修正：半凯利 (Half-Kelly)

股市的胜率和赔率是不确定的。为了安全，成熟交易员通常只使用凯利公式计算值的 1/2 或 1/4 进行开仓。这能极大地降低波动风险。

4.2 金字塔分仓法

  1. 试探仓 (20%-30%)：逻辑确立时买入。如果亏损，说明判断错误，损失有限。

  2. 确认仓 (30%-40%)：只有浮盈才能加仓！ 趋势符合预期时加码。

  3. 决胜仓 (剩余)：趋势完全确立或宏观重大利好时。

4.3 止损与止盈

  • 止损 (Stop-Loss)：
    • 逻辑证伪：买入理由消失（如业绩暴雷）。
    • 技术破位：跌破20日均线或箱体支撑。
    • 硬止损：单笔亏损达 -8% 无条件卖出。
  • 止盈 (Take-Profit)：
    • 左侧：估值过高（PEG > 1.5）分批卖出。
    • 右侧：跌破 10日/20日 均线离场（吃完主升浪）。

第二部分：修正后的投资逻辑闭环 🔄

基于上述知识体系，一个完善的投资逻辑闭环应包含以下六个步骤：

  1. 宏观定势：利用美林时钟判断周期（买股还是买债？）。
  2. 中观选赛道：寻找景气度向上、有定价权的行业。
  3. 微观选龙头：财报排雷，寻找“铲子股”或寡头。
  4. 择时与估值：计算安全边际，好公司也要好价格。
  5. 仓位与风控：半凯利模型定仓位，设定好止损线。
  6. 复盘优化：无论盈亏，记录原因，修正模型。

《地阶功法卷六》每股收益知多少

发表于 2026-01-28 分类于 知乎收藏

本文来源于知乎，作者：MR Dang​

今天谈一个非常重要，非常基础，做投资必须学会的概念，也就是每股收益的计算。

在任何财经类的考试中，每股收益都是必学必考的内容。

我的读者里有很多基础非常过硬的投资者，CPA也不在少数，粗略看了下应该有上千了。

但是也有很多小白，或者新手投资者。

同时我自己对每股收益也有一些不太成熟的看法。

所以今天的文章就深入浅出的说一下每股收益的计算逻辑。

基础概念：

1.归母净利润

净利润咱们经常说，这个好理解。

你开了个超市，年收入100万，扣去进货房租水电人工折旧的成本80万，最后把税一交，净落了15万，这15万就是净利润。

那么归母什么意思呢？

这个母，就是母公司的意思。

你可以理解成”我“

也就是归属于”我“的净利润。

在上面这个例子里，你的超市一开始的时候不是自己一个人开的，你二舅还入了10%的股份，大舅也入了10%的股份。

你其实只有80%的股份，所以最后那个15万，只能拿走15*0.8=12万，剩下的三万要分给大舅和二舅。

这个12万就是所谓的归母净利润。

2.流通在外：

流通在外什么意思呢？

假设你的超市上市了，上市前100股，你有80股，大舅二舅一共20股。

IPO又发了60股给韭菜。

现在一共160股，你有80股，持股比例50%。

上市以后，股价跌跌不休。

你们一合计，这不行啊，开个股东大会。

发了个公告，说公司要回购股票了，一共回购10股。

这10股先放在公司账上，以后用来奖励店长和收银员。

这个时候，流通在外的股票就只有160-10=150股了。

所谓的内外，指的是在不在上市公司账本上。

在公司账上的叫库存股，一般是限制性股份什么的。

不在公司账上，可以在资本市场交易的，叫流通在外。

流通在外的概念不等于流通股本。

(这里比较绕，要多看看)

比如你的80股由于减持规定，上市以后不能立即交易，不是流通股。

但是属于流通在外的范畴，因为它是在大股东账上，而不是在上市公司账上。

判断是否流通在外的黄金标准就是股票持有人是否是上市公司本身。

3.普通股

普通股就是普通投资者拿的股票，咱们讨论的大部分股票都是普通股。

与普通股相对应的概念叫优先股，一般是银行或者一些大型的企业发行的类似于债券的一类东西。

优先股一般不在A股交易，优先股的股东对一般的经营事务没有什么参与权，但是优先股股东可以收取优先股股息。

所谓的优先，意思就是超市破产了，货架和别的盆盆罐罐卖了5万块，这5万块要先赔给债权人，然后是优先股股东，剩下的才能轮到其他普通投资者去瓜分。

4.加权平均股数

按照时间加权计算一年之间平均的股数。

什么意思呢？

比如你年初的时候总股本是100股，7月份的时候IPO了，总股本变成160股了。

那么这一年的加权平均股数就是（100*6+160*6)/12=130股

（6个月100股，6个月160股）

如果是10月才IPO，那加权平均股数=（100*9+160*3)/12=115股

（9个月100股，3个月160股）

基础概念学完了，咱们来学每股收益。

每股收益一般就是投资者俗称的业绩。

这个业绩大约等于净利润/总股本

你平时用这个来做决策，问题不大。

但是有些投资者用这个测算以后，发现和财报上的不一样。

那我们就需要更精确的计算。

每股收益分两种，一种叫基本每股收益，一种叫摊薄每股收益。

投资者一般看重的是前者，后者是考虑可转债或者其他潜在的股本后的每股收益。

所以这篇文章只讲基本每股收益，摊搏每股收益有兴趣的可以自学哈，原理大差不差的。

基本每股收益=（归母净利润-优先股股息-永续债利息)/流通在外普通股加权平均股数

和上面那个简陋的公式对比，会发现差距主要在以下几点

1.分母上，有一个流通在外的修饰词，如果一家公司有回购或者其他库存股，会导致流通在外的股数变少，也就是分母变小，从而使得真实的基本每股收益比粗略计算的更大。

2.分母上，还有一个加权平均的的修饰词，如果一家公司在一年中发生债转股，增发，或者其他股本变动，则会导致加权平均股数＜期末总股本。

很多人都是用净利润/期末总股本，从而使得真实的基本每股收益比粗略计算的更大。

3.分子上，要减去永续债或者优先股的影响，从而使得真实的基本每股收益比粗略计算的更小。

1.归母净利润和少数股东损益。

超市赚钱的时候，大舅和二舅也赚钱很好理解。

但是超市明明都要破产了，大舅和二舅还在赚钱，是不是有点匪夷所思？

投资者在资本市场，最基本的一件事情就是要找准自己的定位以及自己适用的体系。

体系大概分为两类，强者体系和弱者体系。

所谓强者体系，就是你认为自己比市场聪明，你要主动的给投资标的去定价，然后坚信自己的定价比市场上其他的投资者更准确。

总之在你眼里，市场就是一大傻子。

与之相对的就是弱者体系。所谓弱者体系，就是知道自己不如市场聪明，所以跟着市场来，相信市场的定价都是合理的。

涨了是合理的，跌了也是合理的，涨了又跌了也是合理的，跌了又涨了还是合理的。

总之市场就是大爷，说啥都是对的。你要做的就是在大爷开口说话的瞬间立马听它的话，然后等大爷给的奖励。

这两者没有孰高孰低，都能在资本市场挣钱。

明明自己有比较多的资金，有学科优势，有知识储备，有信息优势，有脑力加持，却放弃自己的优势，去搞打板，追涨杀跌这一套。

强者体系和弱者体系的投资者之间是有沟通障碍的。

比如常见的沟通场景就是强者体系的投资者指着一个跌跌不休的股票，说这东西如何如何低估，如何如何好，市场错杀了。

弱者体系的投资者反手就是一句： 涨有涨的道理，市场永远是对的。

一.顶层设计

铝行业的特点是什么？

一言以蔽之，一低，两高。

低门槛，高能耗，高污染。

能耗和污染有多高？我给大家列两组数据感受一下：

一吨铝电耗在13300度左右，是一吨铜冶炼的15倍，是一吨水泥的150倍。

电解铝碳排放占有色整个行业的70%左右，是有色行业的碳排放第一大户。

所以铝和铜的逻辑完全不同，铜的瓶颈是矿，要盯着矿去找企业。

铝的瓶颈是什么？

是电，所以要盯着电去找企业。

了解了铝的行业特点，我们再看祖国母亲的烦恼是什么？

低门槛——无序竞争，疯狂内卷——应对：提高门槛，提高附加值

高能耗——单位能耗高，电解产量高——应对：限制电解铝产量，产能置换

高污染——火电碳排放高——应对：推行绿电（主要是水电）置换火电

限制产能——只是应对痛点的手段，而不是研究铝业的第一性原理。

但是现在很多的投资逻辑方向，是追着这个限制产能的这个点穷追猛打，这个方向你不要去挖掘投资标的了，像YLGF这种涨的都没什么性价比了，你能买到的都在天上飞。

限制产能只是一种手段，我们需要做的是找出能帮助祖国母亲解决两高一低烦恼的办法。

比如——为什么不用国外的电，排放国外的碳，生产我们的铝？

你一旦跳出限制产能的这个逻辑陷阱，一切就豁然开朗了。

联系我们的“安全”投资主线，整个顶层设计就形成了：

安全→能源安全→电解铝两高一低→异国用电/异国排放→相关标的

它是谁？无所谓，能帮助祖国母亲少排放碳，少用电，多产铝就行。

需要的就是一把趁手的工具，谁能干好这个活，谁就可以得到母亲的奖励。

这就是投资铝的第一性原理。

铝从某一种意义上来说，就是固体的电，金属的煤。

二.商业模式

铝行业从头到脚：

铝土矿→氧化铝→电解铝→原铝→各种铝产品（比如铝型材，铝板带，铝箔）→下游厂商

其中因为产能限制的存在，电解铝这部是限制整个行业的瓶颈，所以就商业模式而言，电解铝是2求（比如魏桥被要求接入国家电网）的商业模式，而其他环节是1求，甚至0求的商业模式。

（至于什么是0求，1求，2求，补课链接→地阶功法卷三）

从总量思维来看，在铝土矿→下游厂商的过程中，整个利润总量可以看作是一定的，如果电解铝的商业模式高于其他环节，利润就会往电解铝环节集中。

那么投资的时候，只有两种选择，要么你投只有电解铝的，要么你投全产业链的。

你如果投没有电解铝步骤的标的，就会容易被电解铝这个环节侵蚀利润。

那么在产能限制这个框架下，你能找的就是ZGLY，SHGF，YLGF，TSLY这四家，被众多投资者可以说是翻烂了，研究透了，那这样的地方你会有认知差和信息差么，会有超额收益么？
