投资新手避坑指南之仓位控制

发表于 2026-01-28 分类于 知乎收藏

本文来源于知乎，作者：MR Dang​

抱歉各位，今天来晚了。

主要是给氪金用户配置的某些重仓标的可能较推荐的时候价格变动了一些，然后有些客户还没落地，会有一些沟通上的事情。

这里也提前说一下，有氪金意向的，最好手边是纯现金再预约，资本市场变化莫测，我只能根据当时可立即落地的价格给你配，后期涨了以后落不了地，你的氪金费就算白瞎了。

——————————分割线————————

今天收到了几个粉丝的小纸条，几乎就挺统一，问仓位怎么控制。

其实仓位控制这件事，你往复杂了搞，可以搞一堆条条框框。

但是对新手，根本没法落地。

对于新手来说，要不要止盈？要不要止损？要不要加仓？

都是问题，公说公有理，婆说婆有理。

对此，我的建议是：

事前计划周全，事中严格执行，事后一刀两断。

————————事前计划周全——————————

仓位控制最重要的一环，新手一般你持仓三四只股就可以了，多了精力分散，注意力不集中，少了达不到分散风险的目的。

每只股不超过50%，最少也别低于10%。

没有见哪个在A股开超市的发了财，所以股票并不是越多越好。

同时还得考虑你的交易成本，一笔交易最低5块钱，根据目前的券商佣金费用，一笔交易保单总价低于3000块是容易吃亏的。

以上只是一些小细节，重点不在这里，重点是选标的。

你选3到4只股票是为了分散风险的，结果你选了万科，保利，招商蛇口。

那就没意义了啊，全是地产股，股灾来了，目送三个一起上跌停板。

或者其中一个崩了，把另外两个吓尿了，你的分散没有任何意义。

你说，那简单，我选不是一个行业的就行了，我选万科，东方雨虹，美凯龙。

这看似三个行业，不还是地产上下游么？

不要这样选。

你的三四个股票一定是互不相关的，没有共同利益的，才可以降低风险。

其次，你如果没把握，一定要选股息率高的，你只是个新手，不要搞那些花里胡哨的。

资本寒冬来临的时候，账户里绿油油的一片，只有握在手里的分红能给你带来一丝慰藉。

很多投资者做不好仓位控制，还在半山腰呢，结果仓位打满了。

眼睁睁的看着股票跌下来，抓耳挠腮的。

这个时候只有分红的钱才能让你复投。

如果你当初买的是10块的股票，预期派息5毛钱。

跌了50%，还剩5块，再分5毛，除权就是4.5.

你的5毛就可以增加11%的股份。

股价涨回9块就解套了。

你1分钱没出，只是因为股息的复投，就把成本降到了9块。

这种情况下你的心理是有底的，你知道只要坚守，胜利终将属于你。

假设你事前选择了A,B,C三个标的。

计划仓位60%。

那就各买20%，空40%。

每个股留10%的仓位，股价每下跌5%，就加1%。

三个股票都腰斩的时候还有10%的空仓，选一个股息最高的，梭哈进去。

事实上你选择的就是5%股息的标的，下跌50%意味着股息都达到10%了，这在现在的A股几乎是不可能的（持续分红）。

计划在手，成竹在胸，那么下跌时候加仓就是有勇有谋。

否则，就变成了接下落的飞刀，非死即伤。

——————————事中严格执行——————

你的计划是每下跌5%就加1%，结果一个跌停把你跌急眼了，看着跌停板马上就要开了，你决定孤注一掷，梭哈进去，大不了等下反弹了做个T，再把多加的仓位减掉。

没想到天不遂人愿，本来马上要打开的跌停板突然就被拍的死死的。

你一脸煞白，完了，被活捉了。

这是新手常见的错误，喜欢看盘，容易被盘面上的各种信息迷惑了心智，脑子一热，本来周密的计划就变成了猜大小。

事实上，看盘除了消磨你的时间并不能有任何助益。

既然已经制定了计划，那就严格执行，按照既定价格在开盘前，或者盘中报出去守株待兔即可。

你的交易只需要保证成功率就已经战胜了一大半投资者，也就是想买的时候可以买到，想卖的时候可以卖出去，而不是指望你的手指能给你带来超额收益。

在现在量化满天飞的A股市场，你宁可相信我是秦始皇也别相信能在量化手里虎口夺食。

那么运气好，涨了怎么办？

无论是新手还是高手，逃顶都是非常难的。

所以对新手，最简单的做法就是设定个止盈，我一般建议30%。

你买的股息率有5%，上涨30%，意味着股息不足4%，属于是鸡肋，食之无味，弃之可惜（针对2025年10月的无风险利率而言）

而且你已经赚了30%了，有什么不满意的呢？

止盈以后寻找下一个标的，积小胜为大胜，有何不可？

————————————事后一刀两断——————————

已经交易完成，就不要瞻前顾后。

看见卖出的股票涨到天上，只有两种结果，一种是懊悔不已，黯然神伤，另一种是不甘心，再次追进去，吃个大套。

所以最好的策略是不要去关注它，专注于你手中的标的，寻找下一个投资机会。

已经分手，就不要回头，

该删的删，该留的也别留。

第二部分：Markdown 语法速查

Markdown 是一种轻量级标记语言，让我们专注于写作。以下是常用语法演示。

1. 标题

使用 # 的数量表示标题等级：

一级标题 (通常由 Hexo 自动生成文章标题)

二级标题 (常用作大章节)

三级标题 (常用作小节)

2. 字体样式

  • 加粗：两边加两个星号 **内容**
  • 斜体：两边加一个星号 *内容*
  • 删除线：两边加两个波浪号 ~~内容~~

3. 列表

无序列表 (使用减号 - 加空格)： - 第一点 - 第二点 - 缩进两个空格可以嵌套

有序列表 (使用数字 1. 加空格)： 1. 第一步 2. 第二步 3. 第三步

4. 引用

使用 > 符号： > 这是一个引用段落。 > 知识就是力量。

5. 插入链接和图片

  • 插入链接：[显示文本](链接地址) 例如：我的博客

  • 插入图片：![图片描述](图片链接) 提示：如果不使用图床，可以将图片放在 source/images 文件夹下（需新建），然后引用 /images/图片名.jpg

6. 代码块

这是程序员最常用的功能。

行内代码：使用反引号 包裹。 例如：请执行git status` 命令。

代码块：使用三个反引号包裹，并指定语言。

第三部分 如何使用 Mermaid 绘制公司股权结构图

在进行 A 股公司基本面分析时，一张清晰的股权结构图能帮我们快速理清公司的实际控制人与核心资产分布。通过 Hexo 博客内置的 Mermaid 支持，我们可以直接用代码“画”出结构图。

1. 结构图代码模板

你可以直接复制以下代码到你的 Markdown 文件中测试。这里以三大运营商的典型国资架构为例：

graph TD
    %% 定义节点样式（可选，用于区分不同层级的公司）
    classDef stateOwn fill:#f96,stroke:#333,stroke-width:2px;
    classDef group fill:#eef,stroke:#333,stroke-width:1px;
    classDef listed fill:#fff,stroke:#333,stroke-width:2px,color:#000;

    %% 结构逻辑与持股比例
    A[国务院国资委 - 实际控制人]:::stateOwn -- 100% --> B(电信集团有限公司 - 控股股东):::group
    
    B -- 62.82% --> C{中国电信 A+H股 - 核心上市公司}:::listed
    D[公众股东及其他] -- 37.18% --> C
    
    C -- 100% --> E(天翼云科技有限公司)
    C -- 100% --> F(各省市级分公司)

2. 语法快速拆解

为了方便大家根据不同公司的年度财报进行修改，这里简单拆解一下核心语法：

  • 绘制方向：graph TD 表示自上而下 (Top to Down) 绘制。如果层级不多但平级公司多，可以改为 graph LR (从左到右)。
  • 节点形状：
    • [文字]：矩形，通常代表最终实控人（如国资委、自然人）或公众股东。
    • (文字)：圆角矩形，适合代表集团母公司或下属子公司。
    • {文字}：菱形，我习惯用来高亮我们要分析的核心上市公司主体。
  • 连接线与持股比例：-- 百分比 --> 用于连接上下级节点，并在箭头上直观地标注持股比例。
  • 高亮样式：通过 classDef 定义颜色和边框，再通过 :::样式名（注意是三个冒号）附加到对应节点上，能让股权穿透图的视觉层级一目了然。

> 避坑提示：在使用 Mermaid 时，确保 ```mermaid 和结尾的 ``` 独占一行，且上下保留空行，否则可能会导致解析失败变成乱码。
